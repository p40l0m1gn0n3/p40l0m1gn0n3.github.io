# paolomignone.di.uniba.it (static)

This repo is a **no-build** static website (HTML/CSS/JS) designed to work on GitHub Pages and on the university domain.

## Edit content
- Main page: `index.html`
- Styles: `assets/styles.css`
- Small JS helpers (mobile menu + last updated): `assets/site.js`
- Images used by the new layout: `assets/img/`

## Projects section
The `Projects` section is intentionally empty and contains a commented **card template** you can copy/paste.

## Legacy page
The previous version is kept as `index_legacy.html` for reference.
